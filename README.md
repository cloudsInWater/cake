# Cake

🍰 egg.js + react technology stack = cake


## Development

```bash
# install dependency
yarn

# web page develop
yarn start
# node.js develop, if want to see web page, should run `yarn run build` command before the following command.
yarn run node:dev
```

## Deployment

```bash
# build web page to './server/app/public' folder
yarn run build
# start server
yarn run node:server
```
